# @minddocdev/fontawesome-svg-core - SVG with JavaScript version

> "I came here to chew bubblegum and install Font Awesome 5 - and I'm all out of bubblegum"

[![npm](https://img.shields.io/npm/v/@minddocdev/fontawesome-svg-core.svg?style=flat-square)](https://www.npmjs.com/package/@minddocdev/fontawesome-svg-core)

## Installation

```
$ npm i --save @minddocdev/fontawesome-svg-core
```

Or

```
$ yarn add @minddocdev/fontawesome-svg-core
```
